# Dashboard Concept — Nusantara Electrified

## Core Thesis

**EV Indonesia melaju cepat, tapi keberhasilannya ditentukan oleh kesiapan industri dan kebersihan listrik.**

Dashboard ini bukan sekadar katalog grafik EV. Ia adalah data story tentang satu pertanyaan besar:

> Apakah Indonesia hanya menjadi pasar EV, atau mampu menjadikan EV sebagai transformasi industri dan energi nasional?

## Creative Direction

Visual style: **dark futuristic editorial**, bukan dashboard korporat biasa.

Keywords:

- electric blue
- nickel green
- coal amber
- deep navy/black background
- glassmorphism panels
- glowing route/map lines
- subtle 3D/particle energy effect
- high-contrast editorial typography

## Storyline

### 1. Opening — “Nusantara Electrified”

Hero dengan headline metric:

- BEV 2020 → 2025 growth.
- Jumlah SPKLU dalam dataset.
- Grid emission factor Indonesia.
- Target SPKLU/EV 2030.

Mood: cinematic, ambitious, but with tension.

### 2. EV Melaju Cepat

Visual: line/bar chart pertumbuhan BEV tahunan.

Message: adopsi EV masuk fase akselerasi.

### 3. Infrastruktur Mengejar

Visual: map titik SPKLU + ranking provinsi.

Message: infrastruktur tumbuh, tetapi terkonsentrasi di wilayah tertentu.

### 4. Seberapa Hijau EV?

Visual: interactive emissions slider.

User changes grid factor and sees BEV gCO2/km shift.

Message: EV tidak berdiri sendiri; kebersihan listrik menentukan manfaat iklimnya.

### 5. Dari Nikel ke Mobil Listrik

Visual: value-chain / Sankey-style diagram.

Message: EV adalah strategi industri: nikel, baterai, TKDN, pabrik, lapangan kerja.

### 6. Nickel & Grid Paradox

Visual: split-screen tension cards.

Left: nickel opportunity. Right: LFP risk and coal-heavy grid.

Message: transisi punya peluang dan trade-off.

### 7. Bukan Menolak, Masih Ragu

Visual: barrier cards for harga, charging, baterai, resale value, keamanan, sumber listrik.

Message: masyarakat tidak sepenuhnya menolak EV; banyak yang masih menunggu bukti kesiapan ekosistem.

### 8. Closing — Tiga Syarat

Final synthesis:

1. Infrastruktur charging merata.
2. Grid makin bersih.
3. Industri lokal naik kelas dari assembly ke value capture.

## Recommended UX

- Single-page scrollytelling.
- Sticky narrative panel on desktop.
- Mobile-first fallback: charts become stacked cards.
- Minimal filters, only where useful.
- Every major number has source/caveat popover.
- Dataset methodology drawer at the end.
